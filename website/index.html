Getting Started

Get up and running with your first EpicChain smart contract in minutes

1

Prerequisites

Install Rust nightly and Go for the complete toolchain

# Install Rust nightly
rustup install nightly
rustup default nightly
rustup target add wasm32-unknown-unknown

# Install Go 1.23+
# Visit https://golang.org/dl/
2

Clone & Build

Clone the repository and build the compiler

# Clone the repository
git clone https://github.com/your-org/epicchain-contract-rs
cd epicchain-contract-rs

# Build the compiler
cd epicchain-wasm
go build -o epicchain-wasm .
cd ..
3

Your First Contract

Try the hello-world example to verify everything works

# Navigate to hello-world example
cd examples/01-hello-world

# Build everything
make all

Hello World Contract

A simple contract to get you started

src/lib.rs
use epicchain_contract::prelude::*;

#[epicchain_contract]
pub struct HelloWorld {
    owner: H160,
}

#[epicchain_contract_impl]
impl HelloWorld {
    pub fn deploy(&self, owner: H160) -> bool {
        self.owner.set(owner);
        rt::log("HelloWorld deployed!");
        true
    }
    
    #[safe]
    pub fn get_owner(&self) -> H160 {
        self.owner.get()
    }
    
    pub fn say_hello(&self, name: String) -> String {
        format!("Hello, {}!", name)
    }
}
